## ECurrency

Enum defining all supported currencies and tokens on the GRVT platform.

```typescript
enum ECurrency {
  // Fiat
  USD = 'USD',
  
  // Stablecoins
  USDC = 'USDC',
  USDT = 'USDT',
  
  // Major Cryptocurrencies
  ETH = 'ETH',
  BTC = 'BTC',
  SOL = 'SOL',
  
  // Layer 2 & Scaling Tokens
  ARB = 'ARB',
  OP = 'OP',
  ZK = 'ZK',
  POL = 'POL',
  
  // Exchange Tokens
  BNB = 'BNB',
  
  // DeFi Tokens
  AAVE = 'AAVE',
  LINK = 'LINK',
  PENDLE = 'PENDLE',
  ENA = 'ENA',
  
  // Layer 1 Tokens
  ATOM = 'ATOM',
  ADA = 'ADA',
  XRP = 'XRP',
  XLM = 'XLM',
  SUI = 'SUI',
  TON = 'TON',
  DOGE = 'DOGE',
  
  // Meme Tokens (1000x contracts)
  KPEPE = 'KPEPE',     // 1000PEPE
  KSHIB = 'KSHIB',     // 1000SHIB
  KBONK = 'KBONK',     // 1000BONK
  
  // Meme & Community Tokens
  WIF = 'WIF',
  POPCAT = 'POPCAT',
  PENGU = 'PENGU',
  FARTCOIN = 'FARTCOIN',
  
  // AI & Tech Tokens
  VIRTUAL = 'VIRTUAL',
  AIXBT = 'AIXBT',
  AI16Z = 'AI16Z',
  KAITO = 'KAITO',
  
  // DeFi & Ecosystem Tokens
  JUP = 'JUP',

  BERA = 'BERA',
  VINE = 'VINE',
  UXLINK = 'UXLINK',
```

## Example Usage

```typescript
import { ECurrency } from '@grvt/sdk';

// Specify currency for trading
const currency = ECurrency.USDC;

// Use in deposit operations
await client.deposit({
  currency: ECurrency.USDC,
  amount: '1000'
});

// Use in trading pairs
const pair = `${ECurrency.BTC}_${ECurrency.USD}`;
```
