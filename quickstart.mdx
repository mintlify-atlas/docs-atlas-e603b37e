This guide will help you make your first API call using the GRVT SDK. We'll initialize a client and fetch your funding account summary.

## Prerequisites

Before you begin, make sure you have:

1. [Installed the SDK](/installation)
2. Your API credentials:
   - **API Key**: Your Gravity API key
   - **API Secret**: Your Gravity API secret (for authenticated requests)

<Warning>
  Never commit your API credentials to version control. Use environment variables or a secure configuration management system.
</Warning>

## Step-by-Step Guide

<Steps>

<Step title="Import the SDK">

First, import the necessary classes and types from the SDK:

```typescript
import { GrvtClient, EGrvtEnvironment } from '@grvt/sdk';
```

The SDK exports:
- `GrvtClient` - The main REST API client
- `GrvtWsClient` - The WebSocket client for real-time data
- `EGrvtEnvironment` - Enum for environment selection (PRODUCTION, TESTNET, STAGING, DEV)
- All types and interfaces from `@grvt/client`

</Step>

<Step title="Initialize the Client">

Create a new instance of `GrvtClient` with your configuration:

```typescript
const client = new GrvtClient({
  apiKey: 'your-api-key',
  apiSecret: 'your-api-secret',
  env: EGrvtEnvironment.DEV,
});
```

<Note>
  The `apiSecret` is optional but required for operations that need signing (transfers, withdrawals). For read-only operations like getting account summaries, you only need the `apiKey`.
</Note>

**Environment Options:**

- `EGrvtEnvironment.PRODUCTION` - Live trading environment
- `EGrvtEnvironment.TESTNET` - Public testing environment  
- `EGrvtEnvironment.STAGING` - Pre-production testing
- `EGrvtEnvironment.DEV` - Development environment

</Step>

<Step title="Make Your First API Call">

Let's fetch your funding account summary:

```typescript
const accountSummary = await client.getFundingAccountSummary();
console.log('Account Summary:', accountSummary);
```

This returns information about your funding account, including balances and account details.

</Step>

<Step title="Try More Operations">

Now that you have a working client, try some other common operations:

### Get Sub-Account Summary

```typescript
const subAccountSummary = await client.getSubAccountSummary({
  sub_account_id: 'your-sub-account-id',
});
```

### Get Current Server Time

```typescript
const currentTime = await client.getCurrentTime();
console.log('Server time:', currentTime); // Returns milliseconds since epoch
```

### Execute a Transfer

```typescript
import { ECurrency, ETransferType } from '@grvt/sdk';

const transfer = await client.transfer({
  from_account_id: 'from-account-id',
  from_sub_account_id: 'from-sub-account-id',
  to_account_id: 'to-account-id',
  to_sub_account_id: 'to-sub-account-id',
  currency: ECurrency.USDT,
  num_tokens: '100',
  transfer_type: ETransferType.STANDARD,
});
```

<Note>
  The SDK automatically signs transfer and withdrawal requests using your `apiSecret`. You don't need to manually generate signatures unless you want to customize the signing options.
</Note>

</Step>

</Steps>

## Complete Example

Here's a complete working example that ties everything together:

```typescript
import { GrvtClient, EGrvtEnvironment } from '@grvt/sdk';

async function main() {
  // Initialize the client
  const client = new GrvtClient({
    apiKey: process.env.GRVT_API_KEY!,
    apiSecret: process.env.GRVT_API_SECRET!,
    env: EGrvtEnvironment.DEV,
  });

  try {
    // Get funding account summary
    const accountSummary = await client.getFundingAccountSummary();
    console.log('Account Summary:', JSON.stringify(accountSummary, null, 2));

    // Get current server time
    const serverTime = await client.getCurrentTime();
    console.log('Server Time:', new Date(serverTime).toISOString());

  } catch (error) {
    console.error('Error:', error);
  }
}

main();
```

## Using Environment Variables

For better security, store your credentials in environment variables:

```bash .env
GRVT_API_KEY=your-api-key
GRVT_API_SECRET=your-api-secret
```

Then load them in your code:

```typescript
import { config } from 'dotenv';
config();

const client = new GrvtClient({
  apiKey: process.env.GRVT_API_KEY!,
  apiSecret: process.env.GRVT_API_SECRET!,
  env: EGrvtEnvironment.DEV,
});
```

<Note>
  Make sure to add `.env` to your `.gitignore` file to prevent committing secrets.
</Note>

## WebSocket Quick Start

For real-time data streaming, use the WebSocket client:

```typescript
import { GrvtWsClient, EGrvtEnvironment } from '@grvt/sdk';

const wsClient = new GrvtWsClient({
  apiKey: process.env.GRVT_API_KEY!,
  env: EGrvtEnvironment.DEV,
});

// Connect to WebSocket
await wsClient.connect();

// Subscribe to transfer history
wsClient.subscribeTransferHistory(
  'main-account-id',
  (data) => {
    console.log('Received transfer:', data);
  },
  'sub-account-id' // optional
);

// Disconnect when done
// wsClient.disconnect();
```

<Note>
  The WebSocket client automatically handles reconnection with exponential backoff. Subscriptions need to be manually restored after reconnection.
</Note>

## Troubleshooting

### Authentication Errors

If you receive authentication errors, verify:
- Your API key and secret are correct
- You're using the right environment
- Your API credentials have the necessary permissions

### Type Errors

The SDK is fully typed. If you encounter type errors:
- Make sure you're using TypeScript 5.0 or higher
- Check that you're importing types correctly from `@grvt/sdk`

### Connection Issues

For WebSocket connection issues:
- Verify network connectivity
- Check that you're using the correct environment
- Ensure your API key is valid for WebSocket connections
